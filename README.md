# Clase 12/01/2026
  ¿Que es la inteligencia artificial?
  Es un sistema/herramienta que imita la inteligencia y razonamiento humano
  (un punto interesante seria decidir si solo la inteligencia humana) para
  llevar a cabo la resolucion de problemas.
  "Maximizar la esperanza (expectation) de la utilidad de un problema".
